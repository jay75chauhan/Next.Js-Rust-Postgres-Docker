# Next.Js-Rust-Postgres-Docker

# Backend Rust (no framework, Serde for serialization)

```
cargo new backend
```

# Frontend (Next Js)

```
# npx create-next-app@latest --no-git

# npm run dev

# npm i axios

```

# DB - postgres:12 (docker)

```
# docker exec -it db psql -U postgres

# \l (List of database)

# \dt (relations)

```

# Docker

```
# docker-compose up -d
```
